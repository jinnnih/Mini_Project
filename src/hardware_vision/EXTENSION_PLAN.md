# 🔭 Hardware Vision — 실제 로봇 전환 계획

> 현재 이 프로젝트는 **Gazebo 시뮬레이션** 환경에서 동작합니다.  
> 이 문서는 시뮬레이션을 넘어, 실제 AGV 로봇에 라즈베리파이를 탑재하여  
> 현실 세계에서 동작하는 시스템으로 전환하기 위한 기술 비전을 담고 있습니다.

---

## 💡 전환 배경

본 프로젝트의 모티브는 **현대자동차그룹의 AI 주차 로봇(Parking Robot)** 입니다.  
차량 하부로 진입한 로봇이 차를 정확한 위치로 이동시키는 개념을 응용하여,  
자동세차장 입구에서 차량을 레일 중앙으로 자동 정렬하는 시스템을 목표로 합니다.

시뮬레이션으로 검증된 비전 파이프라인과 PID 제어 로직을,  
**라즈베리파이 + AGV 로봇** 조합으로 실제 환경에 이식하는 것이 최종 목표입니다.

| 구성 요소 | 현재 (시뮬레이션) | 전환 후 (실제 로봇) |
|-----------|------------------|---------------------|
| **연산 장치** | PC (x86_64) | 라즈베리파이 5 (ARM64) |

## 🤖 실제 로봇 구성 계획

### 로봇 플랫폼
- **형태**: AGV(무인운반차) 스타일 저상형 플랫폼
- **역할**: 차량 하부로 진입 → 차량을 밀어 세차장 레일 중앙으로 정렬
- **구동 방식**: 메카넘휠 or 옴니휠 (전방향 이동 가능 → 좌우 정밀 정렬에 유리)

### 탑재 컴퓨터
- **라즈베리파이 5** (4GB 이상 권장)
- 현재 PC에서 실행 중인 `vision_node`, `control_node` 를 그대로 이식
- ROS2 Jazzy를 라즈베리파이 Ubuntu 24.04 (Noble) 에 네이티브 설치

### 카메라
- **Pi Camera Module v3** (라즈베리파이 전용 CSI 연결)
- 로봇 전면 저상부에 장착 → 세차장 기둥을 정면으로 촬영
- 실제 조명 환경에 맞게 HSV 범위 재조정 필요

## 🛠 라즈베리파이 기술 적용 포인트

### 1. ROS2 이식
```
현재: PC에서 vision_node, control_node 실행
전환: 라즈베리파이에서 동일 노드 실행 (ARM64 빌드)
```
- `colcon build` 동일하게 사용 가능
- `cv_bridge` ARM64 빌드 필요
- 소프트웨어 렌더링 환경변수 제거 (`LIBGL_ALWAYS_SOFTWARE` 불필요)

### 2. GPIO 직접 제어
```
현재: 아두이노 시리얼 통신으로 LED 제어
전환: 라즈베리파이 GPIO 핀으로 직접 LED / 부저 제어
```
- `RPi.GPIO` 또는 `gpiozero` 라이브러리 사용
- 정렬 완료 신호를 GPIO HIGH로 출력 → LED 점등

### 3. 모터드라이버 연결
```
현재: Gazebo DiffDrive 플러그인이 /cmd_vel 수신 → 가상 바퀴 회전
전환: 라즈베리파이 GPIO → 모터드라이버(L298N) → DC 모터
```
- `control_node` 의 `/cmd_vel` Twist 메시지를 받아
- GPIO PWM 신호로 변환 → 모터드라이버로 전달
- 새로운 `motor_driver_node.py` 추가 예정

### 4. 카메라 토픽 변경
```
현재: /camera/image_raw (Gazebo 가상)
전환: v4l2_camera 패키지로 Pi Camera → 동일 토픽명 유지
```
- `vision_node.py` 수정 없이 그대로 사용 가능

## 📋 필요 부품 목록 (예상)

| 부품 | 용도 | 비고 |
|------|------|------|
| 라즈베리파이 5 (4GB) | 메인 연산 | ROS2 + OpenCV 실행 |
| Pi Camera Module v3 | 기둥 촬영 | CSI 연결 |
| L298N 모터드라이버 | 모터 제어 | GPIO PWM 입력 |
| DC 기어드 모터 × 4 | 로봇 구동 | 고토크 필요 |
| 메카넘휠 × 4 | 전방향 이동 | 좌우 정렬 정밀도 향상 |
| LED (녹색/적색) | 정렬 상태 표시 | GPIO 직접 연결 |
| 부저 | 정렬 완료 알림 | GPIO PWM |
| 리튬 배터리 팩 | 전원 공급 | 모터/라즈베리파이 분리 전원 권장 |
| 저상형 플랫폼 섀시 | 로봇 본체 | 차량 하부 진입 가능한 높이 |

## 🗺 전환 로드맵

```
Phase 1 (현재 완료)
└── Gazebo 시뮬레이션으로 비전 파이프라인 + PID 제어 검증

Phase 2 (단기 목표)
└── 라즈베리파이에 ROS2 설치 + vision_node, control_node 이식
└── Pi Camera로 실제 기둥 HSV 범위 재조정

Phase 3 (중기 목표)
└── motor_driver_node 작성 → 실제 모터 제어
└── GPIO LED / 부저 연동

Phase 4 (최종 목표)
└── AGV 플랫폼 탑재 → 실제 세차장 환경 테스트
└── 전체 시스템 통합 데모
```
